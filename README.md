# AdventOfCode2021

Create a local pyenv adventofcode which is currently using python version 3.9.7
```
pyenv install 3.9.7
pyenv virtualenv 3.9.7 adventofcode
pyenv activate adventofcode
```

Replace your input files

Run the program:
```shell
python day1/day1.py
python day2/day2.py
python day3/day3.py
```
